# MILSPOON

<hr className='my-8 border-gray-300' />

## 사용 기술

**Frontend**: React Native, React Router, SCSS

**Backend & AI**: Express, MySQL, Teachable Machine

**협업 도구**: Git, GitHub

**배포**: Vercel

**수상**: 2023년 서울디지텍고등학교 국방 공간정보 해커톤 **대상**

## 프로젝트 개요

신병 혹은 입영 대기자를 위한 **국방 지식 교육 어플리케이션**입니다.  
군 생활에 필요한 기초 정보를 쉽고 빠르게 익힐 수 있도록  
인터랙티브 콘텐츠와 AI 훈련 기능을 제공합니다.

## 주요 기능

국방 용어 퀴즈
국방 단어 사전
AI 포병 수신호 훈련 (Teachable Machine 연동)
방공호 위치 확인 (위치 기반 지도)
내 정보
