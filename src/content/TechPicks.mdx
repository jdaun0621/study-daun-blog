# TechPicks content

## 사용 기술

- **Frontend**: Flutter 
- **Backend**: Firebase  
- **협업 도구**: Git, GitHub  
- **배포**: 
- **수상**: BeyondClass **우수상**



## 프로젝트 개요

신병 혹은 입영 대기자를 위한 **국방 지식 교육 어플리케이션**입니다.  
군 생활에 필요한 기초 정보를 쉽고 빠르게 익힐 수 있도록  
인터랙티브 콘텐츠와 AI 훈련 기능을 제공합니다.



## 주요 기능

- 로그인 기능  
- 홈 화면
- 노트북 성능 비교
- 핸드폰 성능 비교
- CPU 성능 비교
