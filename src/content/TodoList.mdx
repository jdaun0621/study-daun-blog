# To do List content

## 사용 기술

- **Frontend**: React, React Router
- **Backend**: Express, 
- **협업 도구**: Git, GitHub  
- **배포**: GitHub.io  




## 프로젝트 개요

개인의 목표를 달성하기위해 할 일을 관리하는 **To-do List 웹사이트**입니다.  




## 주요 기능

- 점수 시스템  
- 쓰레기 드롭
- 게임 오버 조건
- 같은 종류 쓰레기 합치기
